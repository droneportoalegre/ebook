# Nota de Privacidade (LGPD)

Este aplicativo pode registrar dados operacionais de missao, como:
- operador/piloto,
- cliente,
- local da missao,
- observacoes e incidentes.

## Finalidade
Os dados sao usados para organizacao operacional, rastreabilidade e auditoria interna das missoes.

## Armazenamento
Na versao atual (HTML/JS puro), os dados ficam salvos no navegador do proprio usuario (`localStorage`).

## Boas praticas
- Coletar apenas o necessario para operacao.
- Evitar incluir dados sensiveis sem necessidade.
- Exportar e apagar dados localmente quando solicitado.

## Responsavel pelo tratamento
Defina no seu produto final: nome, e-mail de contato e politica completa, conforme seu modelo de negocio.
